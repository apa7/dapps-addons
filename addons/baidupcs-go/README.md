## 使用

1. 登录
    ```
        # 进入程序
        docker exec -it dapps-baidupcs-go /bin/sh

        # 登录网盘账号
        BaiduPCS-Go login

        # 下载一个压缩包文件，其中：/test/cmder.zip为百度网盘中资料的路径
        BaiduPCS-Go d /test/cmder.zip

        # 默认保存位置
        ./dapps/docker/addons/baidupcs-go/downloads
    ```
